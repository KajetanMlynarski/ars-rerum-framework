## 📜 Overview

Welcome to the **Ars Rerum Framework**. This repository serves as the official, open-source knowledge base and theoretical foundation for a new paradigm in physics, mathematics, and Artificial General Intelligence (AGI).

The core of this repository is the **[Codex Rerum](CODEX_RERUM.md)** — a comprehensive, 60+ module architectural blueprint that bridges the gap between fundamental physics, additive number theory, and the engineering of Sovereign AI.

We reject the current "simulation paradigm" of Turing/von Neumann architectures and the dual reductionist myths of the "stochastic parrot" and the "50-bit human". Instead, *Ars Rerum* reformulates computation, intelligence, and ethics as strict physical and thermodynamic processes.
